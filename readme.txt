== Description ==

Author: Visualscope - http://www.visualscope.com/

Description: 

You can easily use responsive Video URL Embed Widget to upload videos to your website using an iframe.

The best thing about this amazing widget is that it is free and it is very easy to use as well. Simply get hold of a URL (eg. youtube.com), that contains the embedded source. Then simply add it to the video field of the video responsive. 

== Installation ==

DEMO - http://tonnexmedia.com/

Here are some useful installation tips you will like to go through.

1. Player Video URL: select the video you would like to put on the frame. Make sure that it is the absolute URL of the video or embed player video URL you are hoping to work on.
Here is an example for a copied absolute URL from Youtube.
<iframe width="560" height="315" src="//www.youtube.com/embed/YvR8LGOUpNA" frameborder="0" allowfullscreen></iframe>

Where the actual URL will look like this -  : //www.youtube.com/embed/YvR8LGOUpNA

So you just have to copy - "//www.youtube.com/embed/YvR8LGOUpNA"

and paste it on the field

2. Box Shadow Display: You can switch it on or off as you like using preferences

3. Add Box-Shadow Color : Use this to pick a color

4. Border Radius Display : You can decide to use a border or not

5. Border Radius Size : The integer values will determine the size of the border

6. Padding Display : You can decide to use padding display or not

7. Padding Size : The integer values will determine the size of the padding

== Frequently Asked Questions ==

What are the minimum requirements to use this widget?

What is padding display?
-The padding clears an area around the content (inside the border) of an element. The padding is affected by the background color of the element.
The top, right, bottom, and left padding can be changed independently using separate properties. A shorthand padding property can also be used, to change all paddings at once.
